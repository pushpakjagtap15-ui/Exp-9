## Theory on Pandas
Pandas is a high-level, open-source Python library providing efficient data structures and analysis tools.  
It is designed for **tabular or heterogeneous data**, similar to SQL tables or Excel spreadsheets.

### Core Data Structures
- **Series:** One-dimensional labeled array  
- **DataFrame:** Two-dimensional, size-mutable, heterogeneous tabular structure with labeled axes  

### Characteristics of Pandas
- **Label-based Indexing:** Access data using descriptive labels  
- **Handling Missing Data:** Integrated support for NaN values  
- **Alignment:** Automatic alignment of data based on labels  
- **Time-Series Functionality:** Tools for date ranges, frequency conversion, and moving statistics  

---

## Data Manipulation Theory

### 1. Data Ingestion and Inspection
- Read data from CSV, Excel, SQL, JSON  
- Inspect data types, memory usage, and statistical summaries  

### 2. Data Cleaning
- **Filtering:** Remove outliers or irrelevant rows  
- **Imputation:** Fill missing values with constants, mean, or median  
- **Deduplication:** Remove redundant rows  

### 3. GroupBy Mechanism
- **Split:** Divide data into groups  
- **Apply:** Perform operations (mean, sum, count)  
- **Combine:** Return results in a structured format  

---

## Advanced Analytical Operations
- **Hierarchical Indexing (Multi-Indexing):** Manage nested categories like Year/Quarter  
- **Reshaping and Pivoting:** Transform data between long and wide formats  
- **Merging and Joining:** SQL-like operations  
  - Inner Join  
  - Left/Right Join  
  - Outer Join
